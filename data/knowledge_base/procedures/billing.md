# Procédure — Litiges de Facturation

## Types de litiges courants

### Double prélèvement
**Cause fréquente** : changement de date de prélèvement lors d'une modification d'offre.

**Procédure** :
1. Vérifier l'historique de prélèvements dans le système
2. Identifier la cause du doublon
3. Si confirmé : initier un remboursement immédiat
4. Délai de remboursement : 5 jours ouvrés (virement bancaire)
5. Envoyer une confirmation par email au client

### Facturation hors-forfait
**Cause fréquente** : appels internationaux, SMS/MMS premium, roaming.

**Procédure** :
1. Consulter le détail de facture (appels, SMS, data)
2. Expliquer clairement ce qui est inclus vs. hors-forfait
3. Proposer un ajustement forfaitaire si le montant est significatif (>15€)
4. Proposer une offre adaptée pour éviter les frais futurs

### Facture anormalement élevée après changement d'offre
**Procédure** :
1. Vérifier la date de prise d'effet du changement
2. Recalculer au prorata si nécessaire
3. Appliquer un avoir si erreur de calcul constatée

## Règles de gestes commerciaux

| Situation | Geste autorisé (Niveau 1) |
|---|---|
| Coupure < 4h | Crédit 1 jour d'abonnement |
| Coupure 4h–24h | Crédit 3 jours d'abonnement |
| Coupure > 24h | Crédit 1 semaine + escalade N2 |
| Double prélèvement | Remboursement intégral du doublon |
| Erreur de facturation | Avoir du montant erroné |
| Client fidèle (>3 ans) | Crédit supplémentaire de 20% |

## Processus de remboursement

```
Agent constate l'anomalie
    → Vérification système de facturation
        → Confirmation doublon/erreur
            → Initiation remboursement (outil Refund Portal)
                → Email de confirmation client
                    → Suivi à 5 jours
```

## Accès aux outils

- **Billing Console** : consultation et modification des factures
- **Refund Portal** : initiation des remboursements (plafond N1 : 200€)
- **Historique prélèvements** : 24 mois consultables
