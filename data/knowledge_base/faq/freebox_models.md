# FAQ — Modèles Freebox

## Freebox Ultra (2024)
- **Débit** : jusqu'à 8 Gb/s descendant, 2 Gb/s montant
- **Technologie** : Fibre optique (FTTH)
- **Wi-Fi** : Wi-Fi 7 (802.11be) — 4x4 MIMO
- **Prix** : 49,99€/mois (engagement 12 mois)
- **Inclus** : TV by Canal, appels vers fixes 100+ pays

## Freebox Pop (2020)
- **Débit** : jusqu'à 5 Gb/s descendant, 700 Mb/s montant
- **Technologie** : Fibre optique (FTTH) ou câble
- **Wi-Fi** : Wi-Fi 6 (802.11ax)
- **Prix** : 29,99€/mois
- **Inclus** : TV Free (plus de 250 chaînes), appels fixes France + DOM

## Freebox Revolution (2011, maintenu)
- **Débit** : jusqu'à 1 Gb/s descendant
- **Technologie** : ADSL, VDSL, Fibre
- **Wi-Fi** : Wi-Fi 5 (802.11ac)
- **Prix** : 39,99€/mois
- **Spécificité** : Disque dur intégré 250 Go, lecteur Blu-Ray

## Freebox Delta (2018)
- **Débit** : jusqu'à 10 Gb/s descendant (fibre dédiée)
- **Technologie** : Fibre optique XGS-PON
- **Wi-Fi** : Wi-Fi 6
- **Prix** : 59,99€/mois
- **Inclus** : Freebox Home (domotique), assistant vocal Devialet

## Questions fréquentes

**Q : Comment savoir quel modèle j'ai ?**
R : Le modèle est inscrit au dos de la box, sur l'étiquette. Il est aussi visible dans votre espace client Free.

**Q : Puis-je changer de modèle sans résilier ?**
R : Oui, c'est une migration. Des frais peuvent s'appliquer selon votre engagement en cours.

**Q : Mon modèle est compatible fibre ?**
R : Tous les modèles depuis la Freebox Revolution supportent la fibre si elle est disponible dans votre zone.
